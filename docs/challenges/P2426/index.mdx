## 项目描述

KubeSphere 是在Kubernetes 之上构建的面向云原生应用的分布式操作系统。在 SG2042/Pioneer Box 上，本题要求在 SG2042/Pioneer Box 上移植 KubeSphere。

## 产出及评分要求

- 本题是移植类赛题。
- 至少需要支持在 Linux 上以 All-in-One 的模式进行安装。

## 验证平台

- SG2042/Pioneer Box
