## 项目描述

KubeSphere 是在Kubernetes 之上构建的面向云原生应用的分布式操作系统。在 SG2042/Pioneer Box 上，本题要求在 SG2042/Pioneer Box 上移植 KubeSphere。

## 产出及评分要求

- 本题是移植类赛题。
- 至少需要支持在 Linux 上以 All-in-One 的模式进行安装。

## 验证平台

- SG2042/Pioneer Box

## 提交说明

1. 提交仓库为 https://github.com/rv2036/rvspoc-P2426-kubesphere
2. 请以 Pull request 的形式，将结果提交到上述仓库中。
   - 以下条件中的「PR」均指代相对应于条件 1 仓库链接的 Pull request。
3. 在创建 PR 时，请提供用于复现移植成果的必要内容（包括源码或二进制程序、配置文件、库以及补丁等）。
4. 在创建 PR 时，请在备注信息栏写明用于验证移植成果的详细说明，例如：
   - 验证平台的 OS 名称、版本和安装说明
   - 验证平台的依赖库安装说明
   - 程序编译及安装步骤
   - 程序运行步骤
   - 程序运行结果
5. **在比赛时间截止前，需通过 PR 的方式对提交内容进行完整开源才算完成本赛题。**
