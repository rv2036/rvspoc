## 项目描述

OceanBase 是一款分布式关系数据库软件。在 SG2042/Pioneer Box 上，移植 OceanBase 数据库；尽可能高的提升相关 Benchmark 的分数。

## 产出及评分要求

- 本题是优化竞速类赛题。
- 给出测试工具 TPC-C 对 OceanBase V4.2.4 的性能测试结果。
- 提供复现结果所需要的所有配置和过程说明。

## 验证平台

- SG2042/Pioneer Box
