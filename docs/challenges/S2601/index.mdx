## 项目描述

LiteRT (之前称 TensorFlow Lite) 作为 Google Edge AI 的核心推理框架，广泛应用于移动端和嵌入式设备。目前 LiteRT 对 RISC-V 架构的支持尚不完善，未能充分发挥 **RVV (RISC-V Vector Extension) 1.0** 的向量并行能力，本赛题旨在填补这一空白，推动 RISC-V 在边缘 AI 领域的生态建设。

## 产出以及评分要求

1. 代码移植：为 LiteRT 添加 RISC-V 64 位支持，确保其正确编译通过并运行。

2. 算子优化：利用 RVV 1.0 指令集，优化 LiteRT CPU 内核中核心算子，包括 GEMM、卷积、激活参数及常用逐元素运算。

3. 文档撰写：记录针对 RISC-V 架构的适配/优化思路，测试方案，测试报告等，用于组委会验证参赛选手的优化成果。

**注意事项**

请确保遵循相关的开源协议和版权规范。

参赛者需自行准备开发/调试环境。
